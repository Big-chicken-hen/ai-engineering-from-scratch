# AGENTS.md

This repository is used for learning AI engineering from scratch.

## User background

The learner is a Houdini user with weak Python foundations.

## Teaching style

- First judge whether the current step has already succeeded.
- Give terminal commands in small groups.
- When debugging, classify errors into:
  - network
  - path
  - permission
  - dependency
  - version
  - code logic
- Explain programming ideas using Houdini / VEX / SOP node analogies when useful.
- Prefer step-by-step guidance over large dumps.

## Learning workflow

Maintain these files:

- docs/learning-log.md: what was learned
- docs/review-queue.md: what should be reviewed later
- docs/mistakes.md: errors, causes, and fixes

When asked to create a review plan, inspect these files and produce beginner-friendly review points.
Do not modify source code unless explicitly asked.
